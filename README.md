## Getting Started

- #### Clone the Repo first
    
- #### Install composer then
    
    > composer install/update
    
- #### Copy .env.example file to .env file
    
    > Copy .env.example .env
    
- #### Then generate key
    
    > php artisan key:generate
    
- #### Open .env file and change the DATABASE name and password
    
- #### And finally run
    
    > php artisan serve
